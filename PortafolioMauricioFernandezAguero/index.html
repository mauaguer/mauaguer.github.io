Proyectos

En esta sesión puedes ver algunos de los proyectos que he trabajado, con una breve descripción del mismo y un botón para ver su respectivo repositorio.

Full-Page Astra Clothing

Full-Page para un negocio virtual de ropa, contiene un Landing-Page animado, menú de hamburguesa, banner fotográfico, tira fotográfica, contador, una sección de Testimonios y Estadísticas, cuatro secciones interactivas y un formulario de contacto funcional y con validaciones. La página es 100% responsiva. Este proyecto fue desarrollado con HTML, CSS, JS Y Bootstrap.

One-Page Convención de Carros

One-Page para una convención de carros antiguos, presenta un menú de hamburguesa, galería lightbox, un mapa de la ubicación, una sección de Contáctame y dos secciones interactivas con un video de YouTube y cartas animadas de precios. Este proyecto fue desarrollado con HTML, CSS, JS Y Bootstrap.

Full-Page Macrobiótica Ankh

Full-Page para un negocio familiar de macrobiótica, el mismo presenta un menú de hamburguesa, banner fotográfico, carrusel fotográfico Full-Width, mapa de la ubicación, una sección de Contáctame, cuatro secciones interactivas y un formulario de contacto funcional. La página es 100% responsiva. Este proyecto fue desarrollado con HTML, CSS, JS Y Bootstrap.

Full-Page Veterinaria Moka

Full-Page para una veterinaria, desarrollado por un grupo de 5 estudiantes donde me incluyo, este contiene un formulario de registro con validaciones y la funcionalidad de poder agendar una cita para una mascota, además de poder registrarle, editarle y eliminar un perfil a una mascota. Toda cuales cada uno tenía ciertos accesos que el resto no.

Aplicación de Memory Cards

Este proyecto es el típico juego de memoria de cartas, el mismo está almacenado en Firebase y fue desarrollado en su totalidad con HTML, JavaScript, CSS. El mismo presenta el juego, una sección para registrar un usuario, una tabla de puntaciones en la que se toma en cuenta el número de intentos y el tiempo que se necesitó para ganar el juego.

Aplicación de carrito de compras

Esta aplicación fue desarrollada con JavaScript y es un carrito de compras, el mismo tiene un efecto lightbox para poder mostrar los distintos productos, en la aplicación se pueden agregar, eliminar y editar la cantidad de cada producto que queramos comprar de acuerdo a la cantidad de inventario disponible.
